# C++ Uppercase & Lowercase Checker

A beginner-level C++ program that checks whether a user-entered alphabet is uppercase or lowercase.  
This project demonstrates character validation and classification using the `<cctype>` library.

---

## Features
- Validates whether input is a valid alphabet  
- Identifies uppercase letters using `isupper()`  
- Identifies lowercase letters using `islower()`  
- Provides clean, user-friendly console output  

---

## Concepts Used
- Character validation with `isalpha()`  
- Uppercase/lowercase detection using `isupper()` and `islower()`  
- Basic input/output handling  
- Conditional logic  

---

## How to Run
1. Save the file as `main.cpp`  
2. Compile using:  
   ```bash
   g++ main.cpp -o char_checker
